# 智联万象 MindVista——AI通识课教学交互式课件

## 单元主题
Chapter 1 从输入到决策——神经网络模型的秘密

## 教学目标
- 让学习者理解：只有输入层和输出层的直接模型只能形成单一线性决策规则。
- 让学习者理解：隐藏层的价值在于构造中间特征。
- 让学习者理解：非线性激活函数是隐藏层真正提升表达能力的关键。

## 教学流程
1. The Challenge：先让学习者预测“一条直线是否足够”。
2. Inputs and Outputs：理解输入层和输出层各自承担的角色。
3. Hidden Units：观察隐藏单元如何表示中间几何测试。
4. Activation：对比激活关闭与开启时的表达差异。
5. Build the Network：手工构造四个中间检测器。
6. Trace a Change：理解局部修改如何影响后续计算。
7. What You Learned：形成完整概念闭环。

## 数字化价值
- 通过拖拽边界和探针点，把抽象结构转成可观察的几何行为。
- 通过激活开关的对比，直观看到“线性堆叠仍是线性”的核心概念。
- 通过构造式沙盒，让学习者理解“组合简单检测器”而非“神经网络会自动解决问题”的准确表述。

## 使用建议
- 适合课堂演示、竞赛答辩展示、学生自主探索。
- 推荐 16:9 桌面全屏运行。
- Teacher mode 用于展示公式和更精细的解释，不建议全程默认开启。
